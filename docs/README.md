# Docs Index

This repository keeps durable project knowledge under `docs/`.

## Layout

- `docs/ai/`
  - shared AI knowledge base
  - overview, workflow, module boundaries, checklists, and durable notes

- `docs/roadmap.md`
  - current repository status
  - completed, next, later, and deferred work

- `docs/operations/`
  - session handoff notes

- `docs/superpowers/`
  - historical planning artifacts from earlier workflows

## Suggested Read Paths

### For humans

1. `README.md`
2. `docs/roadmap.md`
3. `docs/ai/project-overview.md`

### For agents

1. `AGENTS.md`
2. `docs/ai/project-overview.md`
3. `docs/roadmap.md`
4. `docs/ai/modules/importer-boundaries.md`
5. only then a matching playbook, checklist, or handoff note
